## 서포트 벡터 머신
- 서포트 백터 머신은 분류를 위한 기준선을 정의하는 모델이다.
- 즉, 분류되지 않은 새로운 데이터가 나타나면 결정 경계(기준선)를 기준으로 경계의 어느 쪽에 속하는지 분류하는 모델이다. 

<br/>

- SVM은 선형 분류와 비선형 분류를 지원한다.
- 비선형에서 대한 커널은 선형으로 분류될 수 없는 데이터들 때문에 발생했다.
- 비선형 문제를 해결하는 가장 기본적인 방법은 저차원 데이터를 고차원으로 보내는 것이다.(수학적 계산이 필요하기 때문에 성능에 문제를 줄 수 있다)

<br/>

<br/>

- 이러한 문제를 해결하고자 도입한 것이 커널 트릭이다.
- 선형 모델에는 선형(liner) 커널이 있고, 비선형을 위한 커널에는 가우시안 RBF 커널과 다항식 커널이 있다.

<br/>

<img width="564" height="449" alt="image" src="https://github.com/user-attachments/assets/3a664b5e-5962-4c7b-a1bc-7b95c7fcdd86" />

<br/>

- 선형 커널:
  - 선형으로 분류 가능한 데이터에 적용하며 기본 커널 트릭으로 커널 트릭을 사용하지 않겠다는 의미와 같다.
- 다항식 커널:
  - 실제로는 특성을 추가하지 않지만, 다향식 특성을 많이 추가한 것과 같은 결과를 얻을 수 있는 방법이다.
  - 즉, 실제로는 특성을 추가하지 않지만, 엄청난 수의 특성 조합이 생기는 것과 같은 효과를 얻기 때문에 고차원으로 데이터 매핑이 가능하다.
- 가우시안 RBF 커널:
  - 다항식 커널의 확장이라 봐도 좋음
  - 입력 베터를 차원이 무한한 고차원으로 매핑하는 것으로, 모든 차수의 모든 다항식을 고려한다.
  - 즉, 다항식 커널은 차수에 한계가 있느데, 가우시안 RBF는 차수에 제한없이 무한한 확장이 가능하다.
